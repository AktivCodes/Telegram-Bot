# 🤖 ActivCodes Bot

## 📌 Описание проекта

ActivCodes Bot — это многофункциональный Telegram-бот, созданный на Python с использованием библиотеки Aiogram.

Бот сочетает в себе игровые механики, работу с API, систему профилей пользователей и встроенный ИИ-чат.

---

## 🚀 Возможности

### 🎮 Игры

* 🪙 Орёл и Решка
* 🎲 Кубик
* ❓ Квиз

### 👤 Профиль

* Баланс пользователя
* Статистика игр
* Победы и поражения
* Система достижений
* Ранги игроков

### 🌤 Погода

Получение актуальной погоды по названию города через Open-Meteo API.

### 💱 Конвертер валют

Конвертация между:

* USD
* EUR
* RUB
* KZT

### 🤖 ИИ Maboy

Общение с искусственным интеллектом внутри Telegram.

### 🎁 Промокоды

Система активации промокодов с выдачей наград.

### 💾 Сохранение данных

Все данные пользователей сохраняются в JSON-файл и не теряются после перезапуска бота.

---

## 🛠 Используемые технологии

* Python 3
* Aiogram 3
* JSON
* Open-Meteo API
* ExchangeRate API
* AsyncIO

---

## 📂 Структура проекта

main.py — запуск бота

adapter.py — игровые и сервисные функции

handles.py — обработчики сообщений и команд

game.py — игровая логика

weather.py — работа с погодой

currency.py — работа с валютами

states.py — FSM состояния

keyboards.py — клавиатуры

players.json — база данных пользователей

info.json — информация обо мне

ai.py — работа с ИИ

---

## 🎯 Цель проекта

Создать удобного Telegram-бота, который объединяет игры, полезные сервисы и современные технологии в одном приложении.
