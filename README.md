# 🤖 ActivCodes Bot

## 📌 Описание проекта

ActivCodes Bot — это многофункциональный Telegram-бот, созданный на Python с использованием библиотеки Aiogram.

Бот сочетает в себе игровые механики, работу с API, систему профилей пользователей и встроенный ИИ-чат.

---

## 🚀 Возможности

### 🎮 Игры

* 🪙 Орёл и Решка
* 🎲 Кубик
* ❓ Квиз

### 👤 Профиль

* Баланс пользователя
* Статистика игр
* Победы и поражения
* Система достижений
* Ранги игроков

### 🌤 Погода

Получение актуальной погоды по названию города через Open-Meteo API.

### 💱 Конвертер валют

Конвертация между:

* USD
* EUR
* RUB
* KZT

### 🤖 ИИ Maboy

Общение с искусственным интеллектом внутри Telegram.

### 🎁 Промокоды

Система активации промокодов с выдачей наград.

### 💾 Сохранение данных

Все данные пользователей сохраняются в JSON-файл и не теряются после перезапуска бота.

---

## 🛠 Используемые технологии

* Python 3

* Open-Meteo API

# ⚙️ Установка и запуск

## 1. Установите Python

Рекомендуемая версия:

Python 3.11+

Проверка установки:

python --version

---

## 2. Установите зависимости

Откройте терминал в папке проекта и выполните:

pip install aiogram requests

---

## 3. Настройте токен бота

Создайте бота через BotFather и получите токен.

Откройте файл:

main.py

При запуске проекта токен передаётся через параметр командной строки.

Пример:

python main.py --token=ВАШ_ТОКЕН

Например:

python main.py --token=ТОКЕН

py main.py --token=ТОКЕН --debug (режим оператор, смотреть все действия бота в терминале)

---

## 4. Запуск проекта

Обычный запуск:

python main.py --token=ВАШ_ТОКЕН

---

## 5. Режим отладки (DEBUG)

Для отображения логов и диагностики ошибок используйте:

python main.py --token=ВАШ_ТОКЕН --debug

или

python main.py --token=ВАШ_ТОКЕН -debug

После запуска в консоли появится:

🤖 Бот запущен в режиме DEBUG...
